
jupyter_cvlabelling_widget
=====================================

Version: |release|

Jupyter notebook widget that offers assisted labelling of images and videos 


Quickstart
----------

To get started with jupyter_cvlabelling_widget, install with pip::

    pip install jupyter_cvlabelling_widget

or with conda::

    conda install jupyter_cvlabelling_widget
